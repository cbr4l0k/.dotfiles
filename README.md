# dotfiles

Personal configuration files managed with [chezmoi](https://www.chezmoi.io/).

## Usage

```bash
# Apply configurations
chezmoi apply

# Edit a config file
chezmoi edit ~/.zshrc

# Update from this repo
chezmoi update
```

## What's inside

- Shell configuration (zsh)
- nvim configuration
- Git settings
- Various scripts and aliases
- ohter `.config` files that I use daily
